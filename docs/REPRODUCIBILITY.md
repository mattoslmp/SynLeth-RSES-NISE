# Reproducibility and provenance

- All bundled tables identify their source DOI and URL.
- Large third-party matrices are not silently redistributed. Acquisition scripts or official portal instructions are provided.
- The exact DepMap release must be recorded in every run directory.
- GDC manifests are retained before download.
- Missing domains reduce coverage and are not imputed as zero.
- Literature priors are separated from empirical TCGA/DepMap components.
- Every final candidate table should include source file checksums, software version and configuration snapshot.
