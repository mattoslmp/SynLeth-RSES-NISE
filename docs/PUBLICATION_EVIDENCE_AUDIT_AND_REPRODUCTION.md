# RSES-Onco publication evidence audit and reproduction

## Scientific rules

The publication workflow preserves missing values and explicitly distinguishes observed negative evidence, neutral evidence, missing evidence, non-eligibility, insufficient sample and technical/source failure. Missing or non-eligible components are never converted to zero. Overlapping representations of the same publication or original dataset are retained for traceability but cannot receive more than one combined evidence unit.

The complete mathematical definition is generated at:

```text

```

## Document rendering boundary

The repository does not invent article or supplement source documents. When DOCX/LaTeX manuscript sources are present, they must import the registered PNG/PDF/SVG files and exact tables from `article_outputs`. Page separation requirements, including any future Figures S68 and S69, apply only when those figures exist as data-supported registered assets; absent identifiers are not fabricated to satisfy numbering.
